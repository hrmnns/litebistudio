# Reporting

_Last updated: 2026-03-08_

## Purpose

Reporting lets you compose structured report packages from dashboards and widgets and export them for sharing.

## What You Can Do Here

- Create and manage report packages
- Organize report pages
- Reuse dashboard/widget content
- Export reports in supported formats

## How to Use It

1. Open `Reporting`.
2. Create or open a report package.
3. Add pages and content blocks.
4. Review ordering and completeness.
5. Export and validate output quality.

## Tips

- Use consistent naming for report categories and packages.
- Keep export-ready reports small and modular to simplify updates.
- Re-run key source queries before final export.

## Related Settings

Open `Settings > Apps > Reports`:

- `Default author`
- `Default theme color`
- `Show header/footer by default`
- `Include audit appendix by default`

These defaults are applied when creating or exporting report packages.
