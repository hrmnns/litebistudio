# Worklist

_Last updated: 2026-03-08_

## Purpose

Worklist is used to track and prioritize records that need action.

## What You Can Do Here

- Create action items from records
- Manage status transitions
- Apply quick batch updates
- Review due and priority context

## How to Use It

1. Open `Worklist`.
2. Filter items by status/priority.
3. Update tasks individually or in batch.
4. Use comments and due fields for handover clarity.

## Tips

- Keep statuses consistent across teams.
- Use priorities and due dates for transparent triage.
- Clean up completed items regularly to keep focus.
